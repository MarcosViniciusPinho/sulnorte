Sulnorte Serviços Maritímos

Versão 0.3.0
Foram adicionados algumas regras específicas devido ao negócio em si, segue abaixo:
  • Só poderá excluir um Porto se o mesmo não estiver sendo usado em Armador ou em Rebocador (O sistema oculta o botão de exclusão caso o Porto estiver sendo usado).
  • Só poderá excluir um Armador se o mesmo não estiver sendo usado em Rebocador (O sistema oculta o botão de exclusão caso o Armador estiver sendo usado).
